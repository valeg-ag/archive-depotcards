## Install Dependencies

```sh
> poetry install
```

## Run Script

```sh
> poetry run python index.py
```

Ensure that you have your csv-files inside "archive-depotcards/data" directory. Expected files format is like follows:

```csv
;;;;;;;;;
;Номенкл. №;Склад;;;;;;;
;272134;551;;;;;;
;272135;551;;;;;;
;272136;551;;;;;;
```
